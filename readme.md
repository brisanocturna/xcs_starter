## xcs_starter

xuscrus' minimal base project start

### Start project
```
> npm install
```

### Base commands
Run dev
```
> npm run dev
```

Check test
```
> npm run test
```

Check contnius integration
```
> npm run ci
```

Make build
```
> npm run dist
```

### Features
* [EditorConfig](http://editorconfig.org/) write well =D
* [eslint](http://eslint.org/) linter =D
* [webpack](https://webpack.github.io/)  bundle =D
